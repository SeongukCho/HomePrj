6장 프로세스 관리
유닉스 옵션
-e	모든 프로세스의 정보를 출력
-f	프로세스의 자세한 정보를 출력
-u uid	특정 사용자에 대한 모든 프로세스 정보를 출력
-p pid	pid로 지정한 특정 프로세스의 정보를 출력
BSD 옵션 묶어서 사용가능 붙임표로 시작하지 않음 (ex : aux)
a	터미널에서 실행한 프로세스의 정보를 출력
u	프로세스 소유자 이름, cpu 사용량, 메모리 사용량 등 상세 정보를 출력
x	시스템에서 실행중인 모든 프로세스의 정보를 출력
GNU 옵션 붙임표 두개로 시작 (ex : --pid)
--pid PID 목록 : 목록으로 지정한 특정 PID 정보를 출력함

ps 	현재 단말기의 프로세스 목록 출력하기
ps -f	프로세스의 상세한 정보를 출력
ps -e	시스템에서 실행중인 모든 프로세스의 정보를 출력
ps -ef	시스템에서 실행중인 모든 프로세스의 상세한 정보를 출력
ps -u	특정 사용자의 프로세스 목록 출력
ps -fu	더 상세한 정보
ps -p 86261	-p 옵션과 함께 특정 PID를 지정하면 해당 프로세스의 정보를 출력
ps -ef | grep bash

ps a	터미널에서 실행한 프로세스의 정보를 출력
ps au	터미널에서 실행한 프로세스의 상세한 정보를 출력
ps ax	시스템에서 실행중인 모든 프로세스를 출력
ps aux	-ef처럼 시스템에서 실행중인 모든 프로세스에 대한 자세한 정보를 출력
